\subsection*{Question 1 --- Link Budget and Feasibility}
A transmitter operates at $f = 900$ MHz with:
\begin{itemize}[leftmargin=1.2cm]
    \item transmit power $P_t = 20$ dBm,
    \item transmit antenna gain $G_t = 2$ dBi,
    \item receive antenna gain $G_r = 2$ dBi,
    \item distance $d = 2.0$ km.
\end{itemize}

Assume:
\[
N = -105\ \mathrm{dBm}, \qquad \gamma_{\min} = 8\ \mathrm{dB}
\]

\begin{enumerate}[label=\alph*)]
    \item Compute the free-space path loss at 900 MHz.
    \item Compute the received power $P_r$.
    \item Compute the minimum required received power for feasibility.
    \item Is the link feasible?
\end{enumerate}

\subsection*{Question 2 --- Indoor Path Loss}
An indoor link has:
\begin{itemize}[leftmargin=1.2cm]
    \item distance-based path loss of $68$ dB,
    \item three concrete walls causing $8$ dB loss each,
    \item one wooden door causing $3$ dB loss.
\end{itemize}

Transmitter and receiver antenna gains are $0$ dBi, and $P_t = 15$ dBm.

\begin{enumerate}[label=\alph*)]
    \item Compute total path loss.
    \item Compute received power.
    \item If receiver sensitivity is $-80$ dBm, does the link work?
    \item Suggest two simple ways to improve this link without changing bandwidth.
\end{enumerate}

\subsection*{Question 3 --- Frequency Reuse}
Assume a cellular layout with reuse factor $N=4$.

\begin{enumerate}[label=\alph*)]
    \item What fraction of total spectrum is available in each cell?
    \item Compare this with a system using $N=7$.
    \item Explain how changing $N$ impacts:
    \begin{enumerate}[label=\roman*)]
        \item co-channel interference,
        \item capacity per cell.
    \end{enumerate}
\end{enumerate}

\subsection*{Question 4 --- Multi-hop Routing}
Nodes have feasible links and transmission energy costs:
\[
E_{12}=2,\quad E_{13}=5,\quad E_{24}=2,\quad E_{34}=1,\quad E_{14}=7
\]
where node 1 is source and node 4 is gateway.

\begin{enumerate}[label=\alph*)]
    \item List all feasible routes from node 1 to node 4.
    \item Compute total energy for each route.
    \item Identify the minimum-energy route.
    \item Give one advantage and one disadvantage of multi-hop communication.
\end{enumerate}
